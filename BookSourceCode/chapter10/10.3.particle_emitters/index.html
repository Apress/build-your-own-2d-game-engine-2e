Q: To add a particle emitter at the current mouse position
E: To toggle drawing of particle bounds
1: to toggle particle/rigid-shape collisions

Behavior Controls:
  • P: Penetration resolution on/off
  • V: Motion on/off
  • H: Inject random velocity to all objects
  • G: Create new Rectangle/Circle (randomly)
Selected Objet Controls:
  • Left/Right: Sequence through selected object
  • WASD-ZX: Move/Rotate(ZX) selected object
  • YU: Inc/Dec RigidShape size (not the Renderable)
  • Up/Down + MNF: Increase/Decrease Mass/Restitution/Friction
Drawing Controls:
  • C: CollisionInfo 
  • T: Texture 
  • R: RigidShape 
  • B: Primitive Bounds (only when RigidShape shows)
