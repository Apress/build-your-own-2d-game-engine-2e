Behavior Controls:
  • P: Penetration resolution on/off
  • V: Motion on/off
  • H: Inject random velocity to all objects
  • G: Create new Rectangle/Circle (randomly)
Selected Objet Controls:
  • Left/Right: Sequence through selected object
  • WASD-Z/X: Move/Rotate(Z/X) selected object
  • Y/U: Inc/Dec RigidShape size (not the Renderable)
  • Up/Down + M/N/F: Increase/Decrease Mass/Restitution/Friction
Drawing Controls:
  • C: CollisionInfo 
  • T: Texture 
  • R: RigidShape 
  • B: Primitive Bounds (only when RigidShape shows)
