Class: GameServer

GameServer()

Manages the server's side of the game.

This class should only be instantiated on the server (using Node.js).

Users of this API should inherit from this class and overwrite the abstract functions.

Constructor

new GameServer()

Constructs a new GameServer. The HTTP server and the WebSocket server will immediately be started. This GameServer's GameStateContainer will initially be empty.
